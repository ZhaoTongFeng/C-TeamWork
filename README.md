
# 影院管理和售票系统
本项目为 **C#** 和 **软件工程** 课程作业，技术和能力有限，可能存在很多问题，仅供参考。

## 简介

本系统分管理端和客户端两个子系统

管理端帮助影院管理员管理电影数据、档期数据、订单数据

客户端为顾客提供购票服务，

## 功能
* 管理端：
  - 查看电影，档期，订单数据
  - 导入电影
  - 自动排制档期
  - 销售统计

* 客户端：
  - 查看正在上映的电影
  - 查看电影对应的档期
  - 选择座位
  - 付款

*付款功能仅仅做了设计，没有具体实现
